# Person 3 - Movie Service

## API
- `GET /movies`
- `POST /movies`
- `PUT /movies/:id`

## Cập nhật mạng
- Copy `movie-service/.env.example` -> `movie-service/.env`
- Nếu đổi port thì đồng bộ lại ở Gateway của Person 1.

## Chạy
```powershell
cd d:\Code\kienTrucPhanMem\Tuan08\movie-ticket-system\team-projects\person-3-movie-service
npm install
npm run install:all
npm run dev
```

Service chạy tại port `8082`.
